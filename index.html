Drupal8 Fields Default markup

Following up on Field default markup - removing the divitis on D.O ive made this page to document the changes.

1. field

Markup: 

  <div class="field-item ..... field-label-hidden">
  Just a field
  </div>
Just a field

css

  .field-item{ color: red;}

2. a field + label

Markup: 

<div class="field-item field ... field-label-above">
  <div class="field-label">label</div>
  <div>the field</div>
</div>
label
the field

css

  .field-item{ color: red;}
  .field-label { color: blue;}
  .field-item .field-label{ color: purple;}
